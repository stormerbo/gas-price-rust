# Windows 用户安装说明

## 📦 安装步骤

1. **下载安装包**
   - 获取 `.msi` 安装包文件

2. **运行安装程序**
   - 双击 `.msi` 文件
   - 按照安装向导提示操作

3. **首次安装可能需要额外步骤**
   - 如果系统提示缺少 WebView2，安装程序会自动下载
   - 需要联网，下载大小约 100MB
   - 下载完成后会自动继续安装

4. **启动应用**
   - 从开始菜单或桌面快捷方式启动
   - 首次启动会自动创建数据库并爬取油价数据

## ⚠️ 常见问题

### 提示"找不到 WebView2Loader.dll"

**原因**: 系统缺少 Microsoft Edge WebView2 运行时

**解决方案**:
1. 下载 WebView2 运行时：https://go.microsoft.com/fwlink/p/?LinkId=2124703
2. 运行下载的安装程序
3. 重新启动应用

### 安装时提示需要管理员权限

**解决方案**:
- 右键点击 `.msi` 文件
- 选择"以管理员身份运行"

### 应用无法启动

**检查项**:
1. 确认 Windows 版本是 Windows 10 1809 或更高
2. 确认已安装 WebView2（参考上面的解决方案）
3. 检查防火墙是否阻止了应用

## 💻 系统要求

- **操作系统**: Windows 10 1809 或更高版本
- **运行时**: Microsoft Edge WebView2（安装时自动安装）
- **磁盘空间**: 至少 200MB 可用空间
- **网络**: 首次运行需要联网获取油价数据

## 🗑️ 卸载

1. 打开"设置" > "应用" > "应用和功能"
2. 找到"中国汽油价格管理"
3. 点击"卸载"

## 📂 数据位置

应用数据存储在：
```
C:\Users\你的用户名\.gas_price\data\gas_prices.db
```

卸载应用不会删除数据文件，如需完全清除，请手动删除上述目录。
