# 中国汽油价格管理系统 - 用户安装说明

## 安装步骤

1. **打开安装包**
   - 双击 `中国汽油价格管理_安装包_v0.1.0.dmg`
   
2. **运行一键安装**
   - 双击 `一键安装.command`
   - 输入密码（如果需要）
   - 等待安装完成

3. **启动应用**
   - 从启动台或应用程序文件夹打开"中国汽油价格管理"

## 如果应用闪退

如果应用启动后立即闪退，请按以下步骤诊断：

### 方法 1: 查看崩溃日志

1. 打开"控制台"应用（在"应用程序" > "实用工具"中）
2. 在左侧选择"崩溃报告"
3. 查找"中国汽油价格管理"的崩溃日志
4. 将日志发送给开发者

### 方法 2: 使用诊断脚本

1. 打开安装包 DMG
2. 双击 `diagnose_crash.sh`
3. 查看输出的错误信息
4. 将输出截图发送给开发者

### 方法 3: 手动运行查看日志

1. 打开"终端"应用
2. 输入以下命令：
   ```bash
   /Applications/中国汽油价格管理.app/Contents/MacOS/中国汽油价格管理
   ```
3. 查看输出的错误信息
4. 将输出截图发送给开发者

## 常见问题

### 问题 1: 提示"应用已损坏"

**解决方案**: 使用安装包中的"一键安装.command"脚本，它会自动处理此问题。

### 问题 2: 应用闪退

**可能原因**:
- 端口 8080-8090 被占用
- 缺少系统权限
- macOS 版本过低（需要 10.15 或更高）

**解决方案**:
1. 关闭其他可能占用 8080 端口的应用
2. 使用诊断脚本查看详细错误
3. 确保 macOS 版本 >= 10.15

### 问题 3: 窗口显示空白

**可能原因**: 后端服务器启动失败

**解决方案**:
1. 完全退出应用（Cmd+Q）
2. 重新启动应用
3. 如果仍然失败，使用诊断脚本查看错误

## 系统要求

- macOS 10.15 (Catalina) 或更高版本
- 至少 100MB 可用磁盘空间
- 端口 8080-8090 中至少有一个可用

## 技术支持

如果遇到问题，请提供以下信息：
1. macOS 版本
2. 崩溃日志或诊断脚本输出
3. 问题的详细描述

## 卸载

如需卸载应用：
1. 将"中国汽油价格管理.app"拖到废纸篓
2. 删除数据目录（可选）：
   ```bash
   rm -rf ~/.gas_price
   ```
