# React + Ant Design 前端

## 开发环境运行

1. 启动后端服务（在项目根目录）：
```bash
cargo run
```

2. 启动前端开发服务器（在项目根目录）：
```bash
npm run dev
```

3. 访问 http://localhost:3000

## 生产构建

```bash
npm run build
```

构建产物会输出到 `dist` 目录。

## 技术栈

- React 18
- Ant Design 5
- Vite
- React Router
- Axios
- Day.js
- @ant-design/plots (图表)
